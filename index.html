Grid System

Our grid has 12 columns and includes five tiers of predefined classes for building complex responsive layouts. Customize the size of your columns on extra small, small, medium, large, or extra large devices however you see fit.

Extra small
≥400px 		Small
≥600px 		Medium
≥800px 		Large
≥1200px 		Extra large
≥1600px
Class prefix .g-xs- .g-sm- .g-md- .g-lg- .g-xl-

Basic Usage

Add a class="grid" to an element that wraps around your grid cells. Any direct children of that element will be a grid cell. You can add classes to these grid cells to control their width at different screen widths. See the examples below to get a basic understanding of the code required.

Simple Two-Column Layout

Collapses to a single column when the screen width is less than the medium breakpoint (800px).

.g-m-6
.g-m-6

Simple Three-Column Layout

Collapses to a single column when the screen width is less than the small breakpoint (600px).

.g-s-4
.g-s-4
.g-s-4

Simple Four-Column Layout

Collapses to two columns when the screen width is less than the small breakpoint (600px), and to a single column when the screen width is less than the medium breakpoint (800px).

.g-s-6 .g-m-3
.g-s-6 .g-m-3
.g-s-6 .g-m-3
.g-s-6 .g-m-3

Complex Multi-Row Layout

Our grid system doesn’t limit you to a single row. Your content can flow onto multiple rows at different screen widths.

Tip: Test your grid at different screen widths to ensure that things collapse as you expect. If a grid cell won’t fit on a row, it will simply drop down to the next row.

.g-m-8 .g-l-6
.g-m-4 .g-l-4
.g-m-6 .g-l-2
